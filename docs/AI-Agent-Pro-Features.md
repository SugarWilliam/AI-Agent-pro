# AI Agent Pro 功能说明文档

## 文档信息
- **版本**: v8.0.0
- **更新日期**: 2025年

## 1. 系统概述

### 1.1 产品定位
AI Agent Pro 是一款基于多模态AI的智能助手应用，支持多种大语言模型，提供对话、任务管理、计划制定等智能服务。

### 1.2 核心特性
- 多模型支持（DeepSeek、GLM、Kimi、通义千问等）
- 智能SubAgent系统
- RAG知识库集成
- 任务与计划管理
- 多模态输入输出

---

## 2. 功能模块

### 2.1 对话系统

#### 2.1.1 基本对话
- **功能描述**: 用户与AI助手进行文本对话
- **使用方式**: 在输入框输入消息，按Enter或点击发送按钮
- **支持特性**:
  - 多轮对话上下文保持
  - 消息复制、下载、删除
  - 语音播放
  - 重新生成回复

#### 2.1.2 消息操作
- **复制**: 点击消息下方的复制按钮
- **下载**: 支持Markdown、HTML、TXT、JSON、图片、PDF格式
- **删除**: 删除单条消息
- **编辑**: 编辑已发送的消息

### 2.2 模型管理

#### 2.2.1 模型选择
- **功能描述**: 选择不同的AI模型进行对话
- **使用方式**: 点击顶部导航栏的模型选择器
- **内置模型**:
  - Auto（自动选择）
  - DeepSeek Chat
  - DeepSeek R1
  - GLM-4-Plus
  - GLM-4-Flash
  - Kimi
  - 通义千问
  - GPT-4o
  - Claude 3

#### 2.2.2 模型配置
- **API Key设置**: 在设置-模型中配置各模型的API Key
- **自定义模型**: 支持添加自定义模型配置

### 2.3 SubAgent系统

#### 2.3.1 助手选择
- **功能描述**: 选择不同能力的AI助手
- **使用方式**: 点击底部工具栏的助手按钮
- **内置助手**:
  - 通用助手
  - 代码助手
  - 写作助手
  - 数据分析助手
  - 学习导师
  - 翻译专家

#### 2.3.2 助手管理
- **添加助手**: 创建自定义SubAgent
- **编辑助手**: 修改助手名称、描述、系统提示词
- **资源配置**: 为助手关联Skills、Rules、MCP、RAG资源

### 2.4 资源管理

#### 2.4.1 Skills（技能）
- **功能描述**: 预定义的专业能力模板
- **内置技能**:
  - 专业写作
  - 翻译专家
  - 代码助手
  - 数据分析师
  - 学习导师
  - 头脑风暴
  - 内容审校
  - 摘要生成
  - 演示文稿
  - 计划制定
  - 研究助手
  - 设计顾问

#### 2.4.2 Rules（规则）
- **功能描述**: AI回复的行为规范
- **内置规则**:
  - 格式规范
  - 语气风格
  - 安全准则
  - 准确性
  - 示例要求
  - 结构清晰
  - 多模态支持
  - 上下文保持

#### 2.4.3 MCP服务
- **功能描述**: 模型上下文协议服务
- **内置MCP**:
  - 文件系统
  - 网络搜索
  - 计算器
  - 翻译服务
  - 文档解析
  - 图像分析

#### 2.4.4 RAG知识库
- **功能描述**: 检索增强生成知识库
- **内置知识库**:
  - 金融知识库
  - 哲学知识库
  - 文学知识库
  - 嵌入式Linux
  - AI知识库
  - 摄影技术
  - 地理知识库
  - 社科知识库

### 2.5 任务管理

#### 2.5.1 创建任务
- **功能描述**: 创建待办任务
- **使用方式**: 点击底部"任务"模式，然后创建任务
- **任务属性**:
  - 标题
  - 描述
  - 优先级（高/中/低）
  - 截止日期

#### 2.5.2 任务操作
- **完成任务**: 勾选任务完成状态
- **删除任务**: 删除不需要的任务
- **查看任务**: 在任务列表中查看所有任务

### 2.6 计划管理

#### 2.6.1 创建计划
- **功能描述**: 创建项目计划
- **使用方式**: 点击底部"计划"模式，输入计划描述
- **AI生成**: AI自动生成计划步骤和TODO列表

#### 2.6.2 计划执行
- **开始计划**: 激活计划并开始执行
- **继续计划**: 在中断后继续执行计划
- **查看进度**: 查看计划完成进度

### 2.7 设置系统

#### 2.7.1 通用设置
- **主题**: 深色/浅色/自动
- **语言**: 简体中文/English
- **发送快捷键**: Enter/Ctrl+Enter
- **字体大小**: 小/中/大

#### 2.7.2 模型设置
- 查看所有模型配置状态
- 配置各模型的API Key
- 添加自定义模型

#### 2.7.3 资源设置
- 管理RAG知识库
- 管理Skills
- 管理MCP服务
- 管理Rules

#### 2.7.4 助手设置
- 查看所有SubAgent
- 编辑助手配置
- 删除自定义助手

#### 2.7.5 同步设置
- 配置同步服务器
- 设置同步间隔
- 手动触发同步

#### 2.7.6 数据管理
- 导出数据
- 导入数据
- 清除所有数据

## 3. 使用指南

### 3.1 快速开始
1. 打开应用，等待初始化完成
2. 选择AI模型（默认Auto）
3. 选择AI助手（默认通用助手）
4. 在输入框输入消息开始对话

### 3.2 切换模式
- **对话模式**: 普通AI对话
- **任务模式**: 创建和管理任务
- **计划模式**: 创建项目计划
- **创意模式**: 创意生成
- **创作模式**: 内容创作

### 3.3 使用SubAgent
1. 点击底部工具栏的助手按钮
2. 选择适合的助手（如代码助手、写作助手）
3. 助手会自动应用其专业能力

### 3.4 管理资源
1. 进入设置-资源
2. 选择要管理的资源类型
3. 添加、编辑或删除资源

## 4. 常见问题

### 4.1 API配置
**Q: 如何配置API Key？**
A: 进入设置-模型，点击对应模型的配置按钮，输入API Key。

**Q: 支持哪些模型？**
A: 支持DeepSeek、GLM、Kimi、通义千问、GPT-4o、Claude等主流模型。

### 4.2 数据安全
**Q: 数据存储在哪里？**
A: 数据存储在浏览器本地存储中，不会上传到服务器。

**Q: 如何备份数据？**
A: 进入设置-数据，点击"导出数据"按钮。
